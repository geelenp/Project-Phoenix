## Visie

Project Phoenix moet een systeem worden dat jarenlang betrouwbaar draait, eenvoudig te onderhouden is en waarvan elke belangrijke ontwerpkeuze gedocumenteerd is.
Project Phoenix bouwt een volledig nieuw Energy Management System op.

Belangrijkste uitgangspunten:

- Native EVCC (geen Docker)
- Home Assistant blijft het automatiseringsplatform
- EVCC wordt de centrale laadcontroller
- Stabiliteit primeert boven functionaliteit
- Elke belangrijke beslissing wordt gedocumenteerd
